To compile program use the command:

javac ArrayListManipulator.java

To run program use the command:

java ArrayListManipulator


ArrayListManipulor object must be decalred before invoking any methods.

Call methods using ArrayListManipulator object.

number_embed method must be printed because it returns an int.
my_paindrome method must be printed because it returns a string.

All other methods can be invoked directly without printing.
